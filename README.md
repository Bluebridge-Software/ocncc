# Oracle Communications Network Charging and Control (OCNCC) Custom Development

**Oracle Communications Network Charging and Control (OCNCC)** is a real-time, high-performance platform for monetising and managing telecom services, including prepaid, postpaid, and IoT billing. It enables operators to deliver innovative services while ensuring revenue assurance, fraud prevention, and operational efficiency.

This repository is maintained by **Blue Bridge Software** to store and manage custom platform development work for OCNCC, including:
- **Wireshark scripts** for protocol analysis and troubleshooting.
- **Call Agents** for custom call handling and routing logic.
- **Feature Nodes** for extending platform functionality.
- Other bespoke tools and integrations.

---
**Note:** This repository is private and intended for internal use by Blue Bridge Software and its partners.
